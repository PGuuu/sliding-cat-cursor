# Privacy Policy

Sliding Cat Cursor does not collect, sell, transmit, or share user data.

The extension stores only local settings through Chrome extension storage:

- enabled or disabled state
- cat height
- downward sensitivity

The extension uses a content script to display a local image overlay and listen
to pointer movement direction. It does not read, collect, store, or transmit:

- page content
- browsing history
- form data
- account data
- cookies
- personal communications
- location
- financial information
- health information

No analytics, tracking pixels, remote code, or external APIs are used.
